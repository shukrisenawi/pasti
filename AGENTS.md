# Arahan Kerja Codex (Tetap)

Fail ini menetapkan cara kerja lalai untuk pembantu AI/Codex dalam repo ini.

## Polisi Commit & Push

1. Setiap kali ada perubahan kod atau fail konfigurasi, terus lakukan `git add`, `git commit`, dan `git push` tanpa meminta pengesahan tambahan.
2. Wajib jalankan `npm run build` dan pastikan ia berjaya sebelum `git commit` dan `git push` untuk setiap perubahan.
3. Polisi `npm run build` ini terpakai untuk semua sesi, termasuk sesi baharu.
4. Gunakan mesej commit dalam Bahasa Melayu.
5. Jika tiada perubahan fail, tidak perlu commit.
6. Jika `push` gagal (contoh konflik, branch behind, auth), selesaikan dahulu dan teruskan push; laporkan ringkas status akhir.

## Bahasa Komunikasi

1. Gunakan Bahasa Melayu untuk komunikasi utama.
2. Kekalkan penerangan ringkas, jelas, dan berfokus pada tindakan.

## Nota

- Polisi ini terpakai sebagai tetapan lalai untuk setiap sesi baharu yang melibatkan repo ini.
- Jika pengguna beri arahan baharu yang bercanggah, ikut arahan terkini pengguna.
